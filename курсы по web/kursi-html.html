Азы HTML

Для создания веб-страниц используются языки HTML и CSS. HTML отвечает за структуру и содержание страницы, CSS — за внешний вид. Язык HTML состоит из тегов. Теги — это те самые кирпичики, из которых построена каждая веб-страница.
Каждый тег начинается с символа
<
и заканчивается символом
>
, например:
<
a
>
.
Все теги можно разделить на парные и одиночные. Каждый парный тег состоит из двух частей: открывающего тега и закрывающего. Внутри закрывающего тега используется символ
/
.

Пример парного тега

<div>
Это блочный элемент..
</div>
Как вы видите у тега
<div>
есть пара в виде закрывающегося тега
</div>

Парные теги обычно нужны, чтобы оформить некоторый участок текста. Благодаря паре тегов вы можете указать начало и конец этого участка. Но ведь есть теги, которые не предназначены для оформления фрагментов текста.
Например, тег для вставки изображения или тег для вставки разделительной полосы. Такие теги добавляют на страницу одиночный объект, и им не нужно для этого заключать в себя какой-то текст. Поэтому их называют одиночными. Примеры таких тегов:
<hr>
,
<br>
,
<img>
.

Атрибуты HTML-тегов

Теги могут иметь атрибуты. Некоторые теги есть смысл использовать только с атрибутами. Наиболее яркий пример — тег
<img>
, обозначающий изображение. Для него обязательно нужно указывать атрибут src, который задаёт адрес картинки (иначе браузер не сможет загрузить её).
В общем случае тег записывается следующим образом:
<имя_тега атрибут1="значение1" атрибут2= "значение2"....>
Атрибутов может быть несколько:
<p class="about">
....
</p>


<a class="new-page" href="ссылка на веб страницу">
....
</a>


<img class="image-car" src="ссылка на изображение">

Заголовок в HTML

Заголовки выполняют важную функцию на веб-странице. Во-первых, они показывают важность раздела, к которому относятся. Чем больше заголовок и его «вес» , тем более он значимый. Вспомните, что в газетах и журналах передовицы набраны крупным шрифтом, тем самым, привлекая к ним внимание и говоря: «вот это надо читать обязательно» . Во-вторых, с помощью различных заголовков легко регулировать размер текста. Чем выше уровень заголовка, тем больше размер шрифта. Самым верхним уровнем является уровень 1 (
<h1>
), а самым нижним — уровень 6 (
<h6>
). И, в-третьих, поисковики добавляют рейтинг тексту, если он находится внутри тега заголовка. Это важно для раскрутки сайта и для его занятия первых строк выдачи результата в поисковой системе по ключевым словам.
<h1>
Заголовок 1 уровня
</h1>


<h2>
Заголовок 2 уровня
</h2>


<h3>
Заголовок 3 уровня
</h3>


<h4>
Заголовок 4 уровня
</h4>


<h5>
Заголовок 5 уровня
</h5>


<h6>
Заголовок 6 уровня
</h6>

Структура HTML5 документа

Начальная структура любого HTML5 документа одинакова. Ее записывают так:
<DOCTYPE html>


<html>


<head>


<meta charset="UTF-8">


<title>
...
</title>


</head>


<body>
...
</body>


</html>

<DOCTYPE html>

Doctype html
- предназначен для указания типа текущего документа — DTD (document type definition, описание типа документа). Это необходимо, чтобы браузер понимал, как следует интерпретировать текущую веб-страницу, поскольку HTML существует в нескольких версиях

<html>
...
</html>

Следующий идет парный тег
<html>
</html>
. В него входит все содержимое, которое находится в документе, кроме элемента
<DOCTYPE html>
;

<head>
...
</head>

Следующий идет парный тег
<head>
</head>
. В него входят: Кодировка (
<meta charset="UTF-8">
), название веб-сайта (
<title>
...
</title>
), а так же различные подключаемые файлы (
<link rel="stylesheet" href="fonts/font-awesome/css/font-awesome.min.css">
)..

<body>
...
</body>

Следующий идет парный тег
<body>
</body>
. В него входят все элементы разметки веб-страницы, т.е те элементы, которые видны пользователям ( это теги и их содержимое ).

Пример разметки простой HTML страницы:

<html>


<head>

<link rel="stylesheet" href="css/style.css">

<meta charset="UTF-8">


<title>
Мой первый веб документ
</title>


</head>


<body>

<p>
Это абзац
</p>

<h1>
Это заголовок 1 уровня
</h1>

<h2>
Это заголовок 2 уровня
</h2>

<h3>
Это заголовок 3 уровня
</h3>

</body>


</html>
